# SeaLab · Aprenda IA no dia a dia Plataforma aberta de micro-aprendizado sobre IA, feita pela equipe Seazone. Trilhas curtas, exercícios práticos e uma assistente (Bia) para tirar dúvidas. ## O que tem aqui - **3 trilhas de aprendizado**: Claude Code, Claude Cowork, Claude Design - **29 prompts úteis** para usar no Claude (com botão de copiar) - **Bia**: assistente virtual que responde perguntas sobre IAs (Claude, Gemini, ChatGPT, Composio, Skills, MCP, etc.) - **Modo escuro** - **Sem login, sem cadastro** — abre e usa ## Como rodar local ```bash node server.js # abre em http://localhost:3000 ``` Ou abra o `index.html` direto no navegador. ## Estrutura ``` sealab-edu/ ├── index.html # SPA ├── css/style.css # Estilos ├── js/ │ ├── app.js # Lógica principal │ └── modules/ │ ├── state.js │ ├── lessons.js # Conteúdo das trilhas │ ├── prompts.js # 29 prompts úteis │ ├── gamification.js (placeholder) │ ├── leaderboard.js (placeholder) │ └── progress.js (placeholder) ├── assets/ # Logos e mascote └── server.js # Servidor Node.js ``` ## Stack - HTML + CSS + JavaScript puro - Sem build, sem dependências - Lucide Icons via CDN - Google Fonts via CDN ## Licença Conteúdo aberto para aprendizado. Use, adapte e compartilhe.
